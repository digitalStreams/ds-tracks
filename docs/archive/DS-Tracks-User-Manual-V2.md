## 1. About DS-Tracks

DS-Tracks is designed to let presenters play their own music tracks from a USB drive without the risk of inadvertently introducing malware into the station's computer network*.

DS-Tracks is a touch-screen appliance connected to the main audio desk. You can use fader XX to activate the input and set the levels.

DS-Tracks stores your session tracks for later use, though you can delete them.

DS-Tracks is entirely independent of the Radio-DJ station automation system.

> *Malware programs can easily infect USB drives. They can install themselves when the USB drive is plugged into a computer. Later, when that USB is inserted into another computer, the malware infects it. The malware can spread quickly throughout a connected network, causing severe damage. We need to avoid that by isolating the USB from the station network, and DS-Tracks does that.

### 1.1 Copy files to a USB drive

Copy your music tracks onto a USB Thumb Drive.

Plug the drive into the USB port located in the circular USB input area on the studio desk.

<!-- IMAGE: USB drive illustration -->

### 1.2 Supported audio formats

DS-Tracks accepts the following audio file formats:

| Format | File Extension | Description |
|--------|---------------|-------------|
| MP3 | .mp3 | Most common audio format |
| WAV | .wav | Uncompressed audio |
| OGG Vorbis | .ogg | Open source compressed format |
| FLAC | .flac | Lossless compressed audio |
| M4A | .m4a | AAC audio format |

**File size limit:** Each file must be 50MB or smaller.

**File naming tips:**
- Use simple file names with letters, numbers, dashes and underscores
- Avoid special characters such as # & % $ in file names
- If a file with the same name already exists, DS-Tracks will automatically rename it

### 1.3 The DS-Tracks appliance

Touch the screen of the DS-Tracks Player to turn it on. If it does not respond, there is an in-line switch on the power cord on the right-hand side of the player.

<!-- IMAGE: DS-Tracks appliance with touchscreen -->

### 1.4 Switching the main keyboard and mouse to DS-Tracks

You can switch the USB selector to the number 1 position labelled USB. It has four push buttons and is located near the main studio monitor on the mixing desk shelf.

<!-- IMAGE: USB selector switch -->

## 2. First Time User

If this is the first time you have used DS-Tracks, it is simple to create an account and upload music tracks.

### 2.1 Entering your username

Use the keyboard to enter your name. Alternatively, you can use the on-screen keyboard by touching the name field to enter your name.

<!-- IMAGE: Login screen with name input field -->

**Username requirements:**
- Must be at least 3 characters long
- Can contain letters, numbers, dashes and underscores
- First name, last name or full name is fine

### 2.2 Using the on-screen keyboard

Type in your name – either first name, last name or full name is fine.

Touch the "Accept" button when ready.

<!-- IMAGE: On-screen keyboard with name entry -->

### 2.3 Starting the session

The keyboard closes, and you now click the "OK" button to get started.

<!-- IMAGE: Login screen with OK button highlighted -->

### 2.4 Choose files dialog

The choose-files dialog box opens.

Click "Choose Files", navigate to your USB drive and open it to see your music tracks.

<!-- IMAGE: Choose files dialog -->

### 2.5 Select the USB drive and the tracks

Select the tracks to import. This is easier if you use the keyboard and mouse, but with the touchscreen, use a dragging motion to select the files to import.

<!-- IMAGE: File browser showing USB drive contents -->

**Tips for selecting multiple files:**
- Hold Ctrl and click to select individual files
- Hold Shift and click to select a range of files
- Use Ctrl+A to select all files in a folder

### 2.6 Reporting the tracks uploaded

DS-Tracks will show the list of files it has just imported. Click OK.

<!-- IMAGE: Upload confirmation dialog showing imported tracks -->

**Note:** Only valid audio files will be imported. If a file is not a supported audio format, it will be skipped.

### 2.7 Selecting a track to play

Your music tracks are now ready to play. Open the USB fader on the desk and click on the track you want to play.

<!-- IMAGE: Track list with "Select the track to play" annotation -->

### 2.8 Using the DS-Tracks audio player

The track playing is shown in the audio player along with duration etc. The pause and volume buttons are active. The track is highlighted in red in the tracklist.

Clicking the track button again stops the track.

<!-- IMAGE: Audio player showing track playing -->

**Audio player controls:**
- **Play/Pause button:** Start or pause the current track
- **Progress bar:** Shows playback position; click to seek
- **Volume slider:** Adjust the playback volume
- **Track title:** Displays the name of the currently playing track

**Note:** Once the track finishes playing, DS-Tracks will wait until you click on another track, allowing you to use the microphone, play a promo or switch to automated playing using Radio DJ.

You will need to manage the desk volume sliders in this case.

## 3. Returning Users

Returning users will be able to access their previous sessions. Select the "Show current users" link.

### 3.1 Listing previous sessions

Select your name from the list. You may need to scroll if the list is long.

You can also create a new user if your name is not on the list.

<!-- IMAGE: User list dropdown -->

### 3.2 The session list for a user

DS-Tracks will show all of your saved sessions. They will have your user name and the date & time they were created.

<!-- IMAGE: Session list screen -->

There are several options:

| Button | Action |
|--------|--------|
| **Add Tracks** | Upload additional tracks to this session |
| **Show Tracks** | View the list of tracks in the session |
| **Delete (bin icon)** | Remove the session and all its tracks |
| **Create a new Session** | Start a fresh session with new tracks |
| **Home** | Return to the main login screen |

### 3.3 Show session tracks

Clicking "Show Tracks" brings up the list of tracks in the session.

<!-- IMAGE: Session track list popup -->

Click "Close List" to return to the session list.

### 3.4 Choosing new tracks for the session

Clicking "Add Tracks" lets you choose additional tracks to import into the session.

<!-- IMAGE: Choose files dialog for adding tracks -->

### 3.5 Adding more tracks to your session

The new tracks will be automatically added to the session tracklist. You can add other tracks from this screen as well.

The names of the tracks will be updated in the session list for later use.

<!-- IMAGE: Updated track list with new tracks -->

## 4. Auto Play Mode

DS-Tracks can automatically play through your entire list of tracks in sequence. This is useful when you want continuous music playback.

### 4.1 Enable auto-play

DS-Tracks can play the entire list of tracks in a session. Use the "Auto Play" toggle to enable this feature.

<!-- IMAGE: Track list with Enable auto play toggle -->

When auto-play is enabled:
- A "Play All" button appears
- Tracks will play one after another automatically
- You can still click individual tracks to jump to them

### 4.2 Re-order your auto-play tracks

Once Auto Play is enabled, you can re-arrange the tracks to the order you want them to play.

<!-- IMAGE: Track list with drag handles for reordering -->

**Hint:** Putting a number in front of the track name when they are on the USB will force the tracks to import in numerical order. Start at 01, 02, 03 … 09, 10, 11, 12 etc.

The leading "0" forces the computer to correctly sort the first nine digits.

**Example:**
```
01 Opening Theme.mp3
02 First Song.mp3
03 Second Song.mp3
...
10 Tenth Song.mp3
11 Closing Theme.mp3
```

### 4.3 Return to your session list

You can return to your session list by clicking your username. Clicking the Home button takes you to the home screen. You can also add tracks to the current session with the Add Tracks button.

<!-- IMAGE: Player screen with username link highlighted -->

### 4.4 Delete a session

When in the session screen, you can delete an entire session by clicking the bin icon.

<!-- IMAGE: Session list with delete button tooltip -->

### 4.5 Confirm session delete

Click the confirm delete button to delete or cancel if you have changed your mind.

<!-- IMAGE: Delete confirmation dialog -->

**Warning:** Deleting a session permanently removes all tracks in that session. This cannot be undone.

### 4.6 Delete the user

When there is only one session for a user, deleting the last session will also delete that user from the DS-Tracks system.

<!-- IMAGE: Warning dialog about user deletion -->

A warning message will appear to confirm this action. Click OK to proceed or Cancel to keep the session.

## 5. Reports

DS-Tracks includes reporting features to view and export track information. This is useful for music logging and record-keeping.

### 5.1 Accessing Reports

Click the "Reports" button in the bottom-left corner of the main screen to access the reporting features.

<!-- IMAGE: Main screen with Reports button highlighted -->

### 5.2 Viewing all users

The Reports section displays a list of all users who have uploaded tracks to DS-Tracks.

<!-- IMAGE: User list report -->

The list shows:
- Username
- Number of sessions
- Number of tracks

You can sort the list by clicking on column headers.

### 5.3 Viewing user tracks

Click on a username to view all tracks uploaded by that user across all their sessions.

<!-- IMAGE: User tracks report -->

The track list shows:
- Track filename
- Session date
- File location

### 5.4 Exporting track data

You can export the track data for record-keeping purposes. The following export formats are available:

| Format | Description |
|--------|-------------|
| **CSV** | Comma-separated values, opens in Excel or Google Sheets |
| **Excel** | Native Excel spreadsheet format |
| **PDF** | Printable document format |

Click the appropriate export button to download the data.

## 6. Station Customisation (Administrators)

DS-Tracks can be customised to match your station's branding. This section is for station administrators only.

### 6.1 Accessing the customisation panel

To access the customisation panel:

1. Navigate to the admin page (contact your system administrator for the URL)
2. Enter the administrator password
3. Click "Login"

<!-- IMAGE: Admin login screen -->

**Note:** The administrator password should be kept secure and only shared with authorised personnel.

### 6.2 Station information

You can customise the following station details:

| Setting | Description |
|---------|-------------|
| **Station Name** | Full name of your radio station |
| **Station Short Name** | Abbreviated name (used in compact displays) |
| **Website URL** | Your station's website address |
| **Footer Text** | Custom text displayed at the bottom of pages |

<!-- IMAGE: Station information settings -->

### 6.3 Logo customisation

Upload your station's logos to personalise DS-Tracks:

| Logo Type | Recommended Size | Usage |
|-----------|-----------------|-------|
| **Main Logo** | 200 x 80 pixels | Displayed in the header |
| **Tracks Logo** | 150 x 50 pixels | Displayed in the footer |
| **Favicon** | 32 x 32 pixels | Browser tab icon |

<!-- IMAGE: Logo upload settings -->

**Supported image formats:** PNG, JPG, GIF

### 6.4 Colour scheme

Customise the appearance of DS-Tracks to match your station's brand colours:

| Setting | Description |
|---------|-------------|
| **Primary Colour** | Main brand colour (headers, buttons) |
| **Accent Colour** | Secondary colour (highlights, links) |
| **Background Colour** | Page background |
| **Header Background** | Top header area |
| **Text Colour** | Main text colour |
| **Button Colour** | Standard button background |
| **Button Text Colour** | Text on buttons |

<!-- IMAGE: Colour customisation panel -->

**Tip:** Use colours that provide good contrast for readability. Test your colour choices on the touchscreen to ensure they are easy to see in the studio environment.

### 6.5 Saving your changes

After making your customisation changes:

1. Review your changes in the preview area
2. Click "Save Changes" to apply
3. Refresh the main DS-Tracks page to see your updates

<!-- IMAGE: Save changes button -->

**Note:** Changes take effect immediately for all users.

## Appendix A: Troubleshooting

### File won't upload

| Problem | Solution |
|---------|----------|
| File is too large | Ensure the file is under 50MB |
| Wrong file format | Use MP3, WAV, OGG, FLAC or M4A files only |
| Special characters in filename | Rename the file using only letters, numbers, dashes and underscores |

### Audio won't play

| Problem | Solution |
|---------|----------|
| No sound from speakers | Check the USB fader is open on the mixing desk |
| Track won't start | Try clicking the track again, or refresh the page |
| Volume too low | Adjust both the on-screen volume and desk fader |

### Can't find my session

| Problem | Solution |
|---------|----------|
| Name not in user list | Check spelling; try scrolling the list |
| Session missing | Sessions may have been deleted; create a new one |

### Screen unresponsive

| Problem | Solution |
|---------|----------|
| Touchscreen not working | Use the in-line power switch to restart |
| Page frozen | Press F5 or refresh the browser |

## Appendix C: Session Naming

DS-Tracks automatically names your sessions using the following format:

```
Username-YYMMDD-HHMMSS
```

For example: `Peter-230314-172721` means:
- User: Peter
- Date: 14th March 2023
- Time: 17:27:21 (5:27 PM)

This helps you identify when each session was created.
